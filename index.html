Curso 2: Introducción a Python y desarrollo web con Flask

Team Liceo Industrial

Descripción

Descripción resumida:

¡Ahora vamos a hacer un sistema! Para lograrlo usaremos Python y Flask. Trabajaremos en equipo utilizando GitHub para una experiencia colaborativa emocionante.

Descripción larga: Vamos a desarrollar un sistema utilizando Python, un lenguaje de programación orientado a objetos. Usaremos Flask para hacer más eficiente el desarrollo y trabajaremos en equipo con GitHub, una plataforma que facilita la colaboración y el control de versiones.

Competencias

  • Construir códigos eficientes y organizados para resolver problemas.
  • Desarrollar habilidades de programación y pensamiento computacional.
  • Aprender a usar herramientas como Python, GitHub, APIs y Flask.

Objetivos de aprendizaje

  • Crear funciones reutilizables, legibles y mantenibles.
  • Estructurar programas de forma modular y escalable.
  • Entender la programación orientada a objetos (POO).
  • Utilizar Git y GitHub de manera efectiva.
  • Consumir APIs para extraer datos externos.
  • Desarrollar aplicaciones web básicas con Flask.

Infografía

¿Qué es un lenguaje de programación? Es un conjunto de reglas que permiten escribir instrucciones para que las computadoras las ejecuten. Ejemplos incluyen Python, Java, y C++.

¿Qué es Python? Un lenguaje de alto nivel, fácil de aprender, versátil y ampliamente utilizado en desarrollo web, análisis de datos e inteligencia artificial.

¿Qué es GitHub? Una plataforma que permite almacenar y colaborar en proyectos de software, con control de versiones y herramientas para equipos de trabajo.

Python y Flask

Flask es un microframework para Python que permite crear aplicaciones web rápidamente, con características como flexibilidad, simplicidad y extensibilidad. Ideal para proyectos pequeños y medianos.
